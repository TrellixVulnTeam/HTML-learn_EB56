Shift + Alt + A - закометрировать/раскоментировать блок
Ctrl + / - закометрировать/раскоментировать строку
Shift + Alt + UpArrow - копировать строку и вставить сверху
Shift + Alt + DownArrow - копировать строку и вставить снизу
Ctrl + Shift + . - перевести фокус на панель навигации
Ctrl + E - перейти к файл...
Ctrl + M - переключить режим клавиши TAB на перемещение фокуса
Shift + Alt + 0 - переключить вертикальное/горизонтальное расположение окон
Alt + UpArrow - переместить строку вверх
Alt + DownArrow - переместить строку вниз
Shift + Alt + F - форматирование файла (отступы
Ctrl + Shift + [ - свернуть внутренню областьвозле курсора
Ctrl + Shift + ] - развернуть внутренню областьвозле курсора
Ctrl + \ - разбить окно по горизонтали
Ctrl+ d - выделить слово
Ctrl+ b - закрыть/открыть боковую панель
Ctrl + Shift + LeftArrow/RightArrow - выделить слово слева/справа
Ctrl + backspace - удалить слово слева
Ctrl + x - удалить строку
Ctrl + Alt + UpArrow / DownArrow - добавить курсо сверху/снизу (позволяет писать одновременно в нескольких строках, типы переменным)
